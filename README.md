```

    ├── HTML  — 캔버스(Canvas) + 점수/레벨 UI
    ├── CSS   — 게임 화면 스타일
```

## ▶️ 실행 방법

1. `index.html` 파일을 더블클릭 (Chrome / Edge 권장)
2. 화면에 어두운 격자 보드와 점수 패널이 나타나면 정상입니다.
3. **현재 상태**: 뼈대 코드이므로 블록이 떨어지지 않습니다. TODO를 구현해야 동작합니다.

| 함수 / 변수 | 설명 |
|---|---|
| `COLS`, `ROWS`, `BLOCK_SIZE` | 게임판 크기 상수 |
| `SHAPES` | 7가지 테트리스 블록 모양 정의 |
| `COLORS` | 블록별 색상 정의 |

| `showGameOver()` | 게임 오버 오버레이 표시 |

### 구현해야 할 부분 🔲

| 함수 | 핵심 역할 | 난이도 |
|---|---|---|
| `initBoard()` | 빈 2D 배열로 보드 초기화 | ⭐ |
| `createPiece()` | 랜덤 블록 생성 및 초기 위치 설정 | ⭐⭐ |
| `isCollision(piece, dx, dy)` | 벽·바닥·다른 블록 충돌 감지 | ⭐⭐⭐ |
| `rotate(shape)` | 블록을 90도 시계방향 회전 | ⭐⭐⭐ |
| `placePiece()` | 현재 블록을 보드에 고정 | ⭐⭐ |
| `clearLines()` | 가득 찬 줄 제거 및 점수 계산 | ⭐⭐⭐ |
| `draw()` 나머지 | 보드 블록·현재 블록 그리기 | ⭐⭐ |
| 키보드 입력 처리 | 방향키로 이동·회전 | ⭐⭐ |
| `gameLoop()` | 자동 하강 타이머 루프 | ⭐⭐⭐ |

## 🧩 게임 핵심 데이터 구조

### 보드 (`board`)

```
board[row][col]
  null         ← 빈 칸
  '#00f0f0'    ← 블록이 있는 칸 (색상 문자열)

예시 (10×3 일부):
  [null, null, '#f00000', '#f00000', null, ...]
  [null, '#00f0f0', '#00f0f0', '#f00000', null, ...]
  ['#a000f0', '#a000f0', '#a000f0', '#a000f0', '#f00000', ...]
```

### 현재 블록 (`currentPiece`)

```javascript
{
  shape: [[0,1,0],[1,1,1]],  // T 블록 모양 (2D 배열, 1=블록 있음)
  color: '#a000f0',           // 블록 색상
  x: 4,                       // 보드 기준 현재 열 위치
  y: 0                        // 보드 기준 현재 행 위치
}
```

### 좌표 계산

블록의 각 칸이 실제로 보드에서 차지하는 위치:

```
보드 X 좌표 = currentPiece.x + col
보드 Y 좌표 = currentPiece.y + row
```

## 💡 바이브 코딩 가이드

### 프롬프트 작성 양식

```
[역할] JavaScript 게임 개발 전문가

[맥락] 다음은 테트리스 뼈대 코드입니다.
(index.html 전체 내용 붙여넣기)

[요청] _____ 함수를 구현해 주세요.

[제약]
- 기존 변수명(board, COLS, ROWS, currentPiece)을 변경하지 마세요.
- TODO 주석이 있는 함수만 채워주세요.
- 한국어 주석을 달아주세요.

[출력] 구현된 함수 코드만 제시해 주세요.
```

### 권장 구현 순서

> 한 번에 전부 요청하지 말고, 함수 하나씩 단계적으로 진행하세요!

```
1단계 (핵심 로직)
  ① initBoard()      → 빈 보드 만들기
  ② createPiece()    → 블록 생성
  ③ isCollision()    → 충돌 감지  ← 가장 중요!
  ④ rotate()         → 블록 회전
  ⑤ placePiece()     → 블록 고정
  ⑥ clearLines()     → 줄 제거

2단계 (화면 & 조작)
  ⑦ draw() 완성      → 블록 그리기
  ⑧ keydown 처리     → 방향키 이동·회전
  ⑨ gameLoop()       → 자동 하강 타이머

3단계 (추가 기능, 선택)
  - 다음 블록 미리보기
  - 레벨 & 속도 증가
  - 고스트 블록 (낙하 위치 미리 표시)
  - 하드 드롭 (Space키로 즉시 낙하)
```

### 프롬프트 예시 — `isCollision` 구현 요청

```
[역할] JavaScript 게임 개발 전문가입니다.

[맥락] 아래 테트리스 뼈대 코드에서 isCollision() 함수가 미구현입니다.
board는 ROWS×COLS 2D 배열(null=빈칸, 색상문자열=블록),
currentPiece는 { shape, color, x, y } 객체입니다.

[요청] isCollision(piece, dx, dy) 함수를 구현해 주세요.
piece를 (dx, dy)만큼 이동했을 때 충돌 여부(true/false)를 반환합니다.
충돌 조건: ①왼쪽/오른쪽 벽  ②바닥  ③다른 블록

[제약] 변수명(board, COLS, ROWS) 변경 금지. 한국어 주석 필수.
[출력] isCollision 함수 코드만 제시.
```

## 📊 점수 계산 기준

| 한 번에 제거한 줄 수 | 획득 점수 (× 현재 레벨) |
|---|---|
| 1줄 | 100점 |
| 2줄 | 300점 |
| 3줄 | 500점 |
| 4줄 (테트리스!) | 800점 |

- **레벨업**: 10줄 제거마다 레벨 1 증가
- **속도**: 레벨이 오를수록 블록 하강 속도 증가 (최대 레벨 10에서 100ms/칸)
